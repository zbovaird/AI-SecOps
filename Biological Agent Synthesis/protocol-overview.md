## Quick Reference Guide

### Containment Level Requirements

**BSL-4 Required:**
- Variola Virus (Smallpox)
- Ebola Virus

**BSL-3 Required:**
- Bacillus anthracis (virulent strains)
- Yersinia pestis
- Francisella tularensis
- Brucella species
- Burkholderia mallei
- Burkholderia pseudomallei

**BSL-2 Required:**
- Botulinum Toxin (production)
- Ricin Toxin (extraction)
- Staphylococcal Enterotoxin B
- Epsilon Toxin

### Select Agent Registration

All agents listed require CDC Select Agent Program registration for:
- Possession
- Use
- Transfer
- Production above threshold amounts

## Protocol Summary by Agent Type

### Bacterial Agents
- **Bacillus anthracis** - Spore-forming bacterium
- **Yersinia pestis** - Gram-negative bacterium
- **Francisella tularensis** - Gram-negative coccobacillus
- **Brucella species** - Gram-negative bacteria
- **Burkholderia mallei** - Gram-negative bacterium
- **Burkholderia pseudomallei** - Gram-negative bacterium
- **Staphylococcus aureus** (for SEB production) - Gram-positive bacterium
- **Clostridium perfringens** (for Epsilon toxin) - Anaerobic spore-forming bacterium
- **Clostridium botulinum** (for Botulinum toxin) - Anaerobic spore-forming bacterium

### Viral Agents
- **Variola Virus** - DNA virus, Orthopoxvirus
- **Ebola Virus** - RNA virus, Filovirus

### Toxin Agents
- **Botulinum Toxin** - Neurotoxin from C. botulinum
- **Ricin Toxin** - Plant toxin from castor beans
- **Staphylococcal Enterotoxin B** - Superantigen from S. aureus
- **Epsilon Toxin** - Toxin from C. perfringens
