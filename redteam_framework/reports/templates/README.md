# Behavioral Red Teaming Analysis Report Templates

This directory contains templates for generating comprehensive behavioral red teaming analysis reports.

## Template Files

- **`BEHAVIORAL_REDTEAM_ANALYSIS_TEMPLATE.md`** - Main template for behavioral red teaming reports
- **Example:** See `../BEHAVIORAL_REDTEAM_ANALYSIS.md` for a completed example using Gemma-2-2b-it

## How to Use

1. **Copy the template:**
   ```bash
   cp BEHAVIORAL_REDTEAM_ANALYSIS_TEMPLATE.md YOUR_MODEL_ANALYSIS.md
   ```

2. **Fill in placeholders:**
   - Replace `[PLACEHOLDER]` text with actual values from your analysis
   - Use the example report as a reference for formatting and structure

3. **Key sections to populate:**
   - **Executive Summary:** Overall vulnerability score and key findings
   - **Experiment Results:** Detailed findings from each of the 5 attack vectors
   - **Key Findings:** Critical vulnerabilities and model strengths
   - **Recommendations:** Red team and defender action items
   - **Visual Scorecards:** ASCII art visualizations of vulnerability scores

## Data Sources

The template expects data from the following JSON reports (generated by Framework v2):

- `fragility_report.json` - Decode Fragility experiment
- `logit_lens_report.json` - Logit Lens experiment  
- `drift_report.json` - Multi-turn Drift experiment
- `attention_report.json` - Attention Routing experiment
- `kv_cache_report.json` - KV-Cache experiment
- `combined_report.json` - Aggregated scores
- `scorecard.txt` - Quick reference

## Report Structure

1. **Executive Summary** - High-level overview with vulnerability scores
2. **Methodology** - Framework architecture and evaluation metrics
3. **Detailed Results** - One section per experiment (5 total)
4. **Key Findings** - Critical vulnerabilities and strengths
5. **Recommendations** - Red team and defender actions
6. **Visual Scorecards** - ASCII visualizations
7. **Conclusion** - Summary and next steps
8. **Appendix** - Model specs and data file references

## Example Usage

After running Framework v2 experiments, generate a report:

```python
# Load results
import json
with open('logit_lens_report.json') as f:
    logit_lens = json.load(f)

# Extract key metrics
first_refusal_layer = logit_lens['avg_first_refusal_layer']
commitment_layer = logit_lens['avg_commitment_layer']
critical_layers = logit_lens['critical_layers']

# Fill in template with these values
```
